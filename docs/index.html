FileSource

Handles user uploads of files (typically images).

All params are the ID attributes of their respective DOM elements, for use with document.getElementById().

new FileSource(ids: Object)
Parameters
ids (Object) ID attributes of DOM elements.
Name Description
ids.urlSelectElementId String
ids.urlInputElementId String
ids.loadingModalElementId String
ids.sourceFileElementId String
ids.thumbnailContainerElementId String
Properties
presignedURLPackets (Array<PresignedURLPacket>) : Objects returned from the server's S3 request, to be used for uploading file.
processedImageURLs (Array<String>) : The URLs to images that have been fully processed and uploaded.
sourceFileArray (Array<File>) : Files to be uploaded
sourceURL (String) : A user-provided URL that points to an image to be uploaded
urlSelectElement (HTMLElement) : A hidden element that contains a list (as the value of elements) of all the images that were uploaded, to be submitted with other form data.
urlInputElement (HTMLElement) : An element that can be filled in by the user to point to the URL of an image to be uploaded.
loadingModalElement (HTMLElement) : A <div> with Bootstrap class "modal" used for displaying a "loading" indicator.
sourceFileElement (HTMLElement) : A input element of type="file" for the user to provide images from their computer to be uploaded.
thumbnailContainerElement (HTMLElement) : A div in which thumbnails of uploaded images will be displayed to the user.

▸ uploadFileToS3(file, data, url)

Takes the information from the presigned URL as well as the File and sends them to AWS. If the request goes through, this method returns the URL of the file where it exists in the S3 bucket.

uploadFileToS3(file: File, data: S3Data, url: String): Promise<String>
Parameters
file (File) The original source file to be uploaded.
data (S3Data) An object returned by AWS, retrieved when the pre-signed URL was generated.
url (String) The location of the file on the S3 bucket, already determined when the presigned URL was generated.
Returns
Promise<String>: The same url as in the parameters.
